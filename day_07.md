# Day 07 – Beginner Linux Commands

Today I practiced some basic Linux commands that help with navigation, file management, and viewing system information.

## pwd
pwd

Prints the current working directory.  
It shows where you are in the file system.

Example:
pwd

## whoami
whoami

Displays the current logged-in user.

Example:
whoami

## date
date

Shows the current system date and time.

Example:
date

## cal
cal

Displays the calendar of the current month.

Example:
cal

## history
history

Shows the list of previously executed commands in the terminal.

Example:
history

## clear
clear

Clears the terminal screen.

Example:
clear

## echo
echo Hello Linux

Prints text to the terminal.

Example:
echo Learning Linux

## touch
touch file.txt

Creates a new empty file.

Example:
touch notes.txt

## mkdir
mkdir myfolder

Creates a new directory.

Example:
mkdir practice

## rmdir
rmdir myfolder

Deletes an empty directory.

Example:
rmdir practice

## Summary

Today I learned beginner Linux commands used to:
- check current directory
- identify the current user
- view date and calendar
- create files and folders
- remove empty directories
- print messages in the terminal
