# Day 02 – File Viewing Commands

## cat
Displays the content of a file.

## less
Views file content page by page.

## more
Displays file content one screen at a time.

## head
Shows the first few lines of a file.

## tail
Shows the last few lines of a file.

## wc
Counts lines, words, and characters in a file.

## sort
Sorts the content of a file.

## uniq
Removes duplicate lines from sorted output.

## grep
Searches for a pattern in a file.

## echo
Displays text or writes text to a file.
